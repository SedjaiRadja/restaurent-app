# Restaurant App

A modern and responsive restaurant web application built using React. This project showcases a clean UI for displaying menu items, along with essential pages like About and Contact.

## 📌 Overview

This is a front-end web application designed for a restaurant.
It allows users to explore the menu, view dishes, and navigate between different sections بسهولة.

The project was built individually and focuses on UI/UX and component-based architecture using React.

## 🚀 Features
- 🧭 Smooth navigation between pages
- 🍔 Menu section with food items
- 📄 About Us page
📞 Contact page
- 💡 Clean and responsive design
- ⚛️ Built with reusable React components
- 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (ES6)
 -React

### To run the project locally:

- npm install
- npm run dev

 * Then open your browser and go to:

- http://localhost:5173

- ⚠️ If npm run dev doesn't work, make sure you're using Vite or check your package.json scripts.
